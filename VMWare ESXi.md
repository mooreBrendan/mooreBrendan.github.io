# VMWare ESXi

## Purpose
VMWare ESXi is a way to run multiple VMs on a server.  VM stands for Virtual Machine.  Virtual machines are computers than run as software on another computer (thus virtual).  This allows you to quickly create multiple servers without having to get new physical devices.  Additionally, you can specify the amout of CPU cores, RAM, and hard drive space for the machines so that you can have more optimized setups for each computer.

# NAS
NAS (Network Attached Storage) is a way of having file storage that is able to be accessed by multiple computers. The NAS I use is a Synology.

## RAID
RAID (RAID has multiple definitions, either Redundant Array of Independent Disks or Redundant Array of Inexpensive Disks) is a technology that allows for error correction.

There are multiple levels of RAID that offer different levels of features.
